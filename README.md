# Todo List
## A Simple Todo List Using Mern Stack .


## Api dependencies 

| dependencies              | Version     |
| ----------------          | ----------- |
| Express.js                | 4.17.1      |
| Mongoose                  | 6.0.14      |
| Cors                      | 2.8.5       |



## devDependencies

| devDependencies           | Version     |
| ----------------          | ----------- |
| Nodemon                   | 2.0.15      |


## Scripts

```json
"scripts": {
    "start": "nodemon app.js",
  }
```
## Installation

```shell
$ git clone Weather App # or clone your own fork
$ cd weather-app
$ cd api
$ npm install # or npm i
$ cd ..
$ cd client
$ npm install # or npm i

```
## Run Api
```shell
$ cd api
$ npm run start
```
## Run Client
```shell
$ cd client
$ npm run start
```
